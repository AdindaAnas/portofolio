Overview

A 3-notebook project analyzing Indonesia's electricity consumption using official BPS data. Covering 34 provinces from 2011–2024, the project integrates provincial consumption trends, socioeconomic drivers, and ARIMA forecasting for national demand through 2030.

Notebook 01 — Consumption Insights (2011–2024)

Dataset: 34 provinsi + agregasi nasional · Cakupan 2011–2024 (data 2016 tidak tersedia)

Key Findings:

  • Pulau Jawa mendominasi — Jawa Barat menjadi provinsi dengan konsumsi tertinggi
  • Konsumsi listrik nasional tren meningkat konsisten, dengan sedikit penurunan di 2020
  • Pasca 2020, pertumbuhan kembali stabil di kisaran 5–7% per tahun
  • Maluku Utara memiliki CAGR tertinggi (~17%), diikuti provinsi-provinsi Sulawesi

K-Means Clustering — 3 Segmen Provinsi:

High Growth
CAGR ~10.97% · konsumsi menengah · didominasi luar Jawa
Moderate Growth
CAGR ~7.04% · pertumbuhan stabil · konsumsi menengah
Mature High-Demand
CAGR ~4.42% · konsumsi tertinggi · didominasi Jawa

Notebook 02 — Socioeconomic Drivers (2017–2024)

Dataset: 272 observasi provinsi–tahun · integrasi data listrik, populasi, PDRB, kemiskinan

Correlation Analysis:

  • Konsumsi listrik berkorelasi sangat kuat dengan populasi (r = 0.90) dan PDRB (r = 0.91)
  • Konsumsi per kapita berkorelasi positif dengan PDRB per kapita (r = 0.68)
  • Tingkat kemiskinan berkorelasi negatif dengan konsumsi per kapita (r = −0.49)

Notebook 03 — ARIMA Forecasting (2025–2030)

ADF Test menunjukkan data tidak stasioner (p-value = 0.994) → digunakan d=1. Walk-Forward Validation dilakukan sebelum fitting ARIMA(1,1,1) pada keseluruhan data. Missing data 2016 ditangani dengan linear interpolation.

Model Performance: MAE ~7,700 GWh · RMSE ~7,800 GWh · Error relatif ~3% dari nilai aktual
